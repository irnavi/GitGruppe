Домашняя работа

1.Заходим в GitHub, выбираем репозиторий
  Нажимаем на  Code
  Выбираем SSH
  Копируем ссылку ключ
  Заходим в  GitBush, выбираем нужную папку
  Вводим git clone, вставляем ссылку, enter
 

2.Чтобы получить последние изменения, нужно выполнить 
  команду git pull. Она скачает и объединит
  все изменения, которые были внесены в репозиторий
  с момента последнего обновления.


3.Команда git status показывает текущее состояние
  рабочей директории и индекса Git. Она отображает,
  какие файлы были изменены, добавлены или удалены в
  рабочей директории, которые еще не были зафиксированы
  в Git. Она также показывает, какие файлы были добавлены
  в индекс, но не зафиксированы, и какие файлы были изменены,
  но не добавлены в индекс.

  Команда git log отображает список коммитов в репозитории.
  Эта команда показывает историю изменений, включая
  информацию о том, кто внес изменения, когда они были
  внесены и сообщение коммита. Команда может быть настроена
  с различными опциями для отображения информации о коммитах
  в более подробном формате.

  Команда git remote -v отображает  связан ли репозиторий с 
  GitHub
